# Thesis Audit

## Vision

A user proposes a theory, and the app researches it. The response is organized into "supporting" and "contradicting" signals. The user isn't given a yes/no or true/false kind of answer; rather a collection of data and analysis that help the user assess their original theory.

Since this is based on the Lovelace KG, this is heavily "financial services" focused.
So a high-level summary of the app is something like "applying the scientific method to understanding the FSI space".
